# FastTool

Modern bilingual (EN/ZH) FastTool website built with Next.js App Router.

## Tech Stack

- Next.js 14 (App Router)
- React + TypeScript
- Tailwind CSS
- shadcn-style UI components

## Run

```bash
npm install
npm run dev
```

## Routes

- `/en`
- `/zh`
- `/en/tools/compress-image`
- `/zh/tools/compress-image`

## Notes

- Main image processing is browser-side (Canvas API + browser-image-compression).
- `remove-background` is a placeholder page for future implementation.
